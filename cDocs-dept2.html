Sorting Department Order

Pressing the "Manage" button under the "Department" submenu while in the Settings view opens the "Department Manager" dialog which allows you to sort and delete departments. You may sort and/or delete multiple departments before saving.

Holding the department anchor allows you to drag a department from it's current list position and drop it into a new position.

The "Save" button saves all changes made to the department manager list as to sorting and deleting, then returns you to the Settings menu.

Pressing the "Cancel" button returns you to the Settings menu without applying any changes.
